# Portfolio

## Overview
This site presents selected experience, technical skills, and project work across:

* Hybrid RAG & retrieval-augmented generation systems
* Agentic AI & LLM task automation
* Multimodal deep learning & predictive modeling
* Production ML pipelines & deployment workflows

## Features
* Animated minimalist hero section with location metadata
* Interactive About card featuring custom mouse parallax & planetary orbital visual effects
* Categorized technical skills, timeline experience, education, recognition, and certifications
* Fully responsive layout adapting dynamically to dark and light modes
* React-based UI built with Vite and Tailwind CSS
* Deployed to GitHub Pages

## Tech Stack
* **Framework**: React 18
* **Build Tool**: Vite
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion
* **Typography**: Google Fonts (Playfair Display, Inter, JetBrains Mono)
* **Icons**: React Icons (Heroicons v2, Simple Icons, Font Awesome)

## Local Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build
```bash
# Build production bundle
npm run build
```

## Deployment
The repository includes a GitHub Actions workflow that automatically builds the site and deploys it to GitHub Pages on every push to the `main` branch.
